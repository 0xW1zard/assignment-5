
### 1️⃣ var, let, and const

We don't use **`var`** anymore because it causes bugs. We use **`let`** for things that change (like a counter) and **`const`** for things that stay the same. It keeps our code safe.

### 2️⃣ The Spread Operator (`...`)

We use `...` to quickly copy or merge arrays and objects. It's the easiest way to move data into a new list.

### 3️⃣ map(), filter(), and forEach()

* **`forEach()`**: We use this to just run a function on every item.
* **`map()`**: We use this to change a list into a new one.
* **`filter()`**: We use this to skip items we don't need.

### 4️⃣ Arrow Functions

These are just shorter functions using `=>`. They make our code look clean and save us from writing the `function` and `return` keywords every time.

### 5️⃣ Template Literals

`We use backticks (``) and `${}` to put variables inside strings. No more messy plus signs (`+`)—it makes our text much easier to read.`
